## IP 地址是什么

电脑中硬件涉及到的地址有两类，分别是 IP 地址与 MAC 地址。

-   **IP 地址**：指逻辑地址，并不唯一，可以根据实际情况进行更改。
-   **MAC 地址**：指硬件地址，具有全球唯一性，并不可以进行更改（软件仿冒并不属于更改）。

那么，究竟什么是 IP 地址呢？

IP 地址指的是互联网协议地址，全称为 Internet Protocol Address。IP 地址将会给网络上的设备、主机电脑分配逻辑上的一个标识符，用以区分不同地域的设备。IP 地址工作在网络协议中的网络层，MAC 地址则工作在数据链路层。

当前主流使用的是 IPv4 这个版本，由 32 位的二进制数组成。IPv4 地址一般由四个两位或三位数字组成（数字是 0 – 255），中间用句号隔开，例如我们比较常见的路由器网关地址 192.168.1.1。

IP 地址主要用于网络设备、服务器、用户终端入网设备的设置。通过 IP 地址来实现不同设备之间的数据通信，明确需要访问的具体设备。

例如，运营商的核心路由器、汇聚交换机、接入层设备等均需要设置 IP 地址；我们接触较多的网站服务器、DNS 服务器等也需要设置 IP 地址；个人电脑、平板、手机也需要通过 IP 地址实现上网；个人电脑上的每个网卡均需要设置一个 IP 地址；无论是有线网卡、无线网卡，甚至是虚拟机的虚拟网卡均要设置 IP 地址。

## IP 地址的工作原理

每个域名都将会与特定的 IP 地址关联。这些域名通过域名系统(DNS)转换为 IP 地址。你可以把这个系统想象成一个巨大的地址簿。

当用户在网页浏览器中键入域名时，ISP 就会执行所谓的域名服务器查询。将查找与域名相关联的域名服务器，然后将用户转发到页面被请求的网站。这看似复杂的过程，一瞬间就可以发生。

IP 地址好比人们的家庭住址。你需要一个包裹送到 A 的家，但不能把 A 的家的真实地址写在包裹上，所以你写上了对应地址的域名，由 ISP 查询后运送包裹。IP 地址就是完整的家庭地址，而域名就像是“A 家地址”的缩写。

## IP 地址的分类

IPv4 地址在设计之初，仅供实验室进行使用，并未料想会如此的普及。因此，IPv4 地址匮乏的事情一直困扰着人们。

于是出现了公网、私网地址的分类：

-   **公网地址**主要用于互联网核心设备、服务器等广域网的设备进行使用。它允许将家庭计算机连接到 ISP，然后 通过与托管它们的服务器通信来访问世界各地的其他网站。
-   **私网地址**主要用于局域网设备使用，不同的局域网可以使用相同的私网地址。这种类型的 IP 地址可以让你做一些事情，比如将家庭计算机连接到无线打印机。每个设备都有自己独特的 IP 地址，可以用来与你的独立的网络中的家庭路由器通信。

这样，在一定程度上缓解了 IP 地址不足的问题。

## 为什么要隐藏 IP 地址

很多人隐藏其 IP 地址的主要原因是可以下载未经允许的非法资料，还有人会出于一系列不同的原因来隐藏 IP 地址，让我们来了解一些最为常见的原因。

**地理限制和审查制度**：例如大家都在使用的科学上网，要是你可以隐藏真实 IP 地址并使它看起来像是正在从另一个区域浏览，则可以绕开这些限制并查看被阻止的网站。例如对于某些流媒体或下载服务，会阻止你访问其他地区的媒体库或下载种子，而隐藏你的 IP 地址可以帮助你摆脱这些限制，随心所欲地访问互联网上的任何网站。

**保护隐私并防止个人信息被滥用**：每当你访问网站时，连接的服务器都会记录你的 IP 地址，并将其附加到该网站可以了解的所有其他数据上：你的浏览习惯，点击内容，浏览特定页面的时间。然后，他们会将这些数据出售给广告公司，广告公司则使用这些数据为你量身定制广告。这就是为什么互联网广告个性化的原因。美国一半的互联网人口只能选择一种 ISP，因此对于许多人来说，要么被监视，要么就没有互联网。

**隐藏真实位置**：即使关闭了位置服务，你的 IP 地址也可以用于跟踪你的位置。拥有你 IP 地址的任何人都可以做到这一点，尽管它不会将你的真实家庭住址或姓名透露出来，但是只要能够访问你的 ISP 客户数据的人都可以轻松找到你。

**规避政府审查**：某些政府会对其国民的线上活动进行监测和审查。政府监测到的活动可能会用于证明你的反政府野心，可能会导致对你作出刑事指控或者更为恶劣的后果。对此你应当做的是无论何时，只要你上网就应当隐藏你的 IP 地址。

**防护黑客攻击**：就像你的互联网服务供应商，搜索引擎，政府可以使用你的 IP 地址来追踪你的线上活动一样，黑客们也可以。并且这些黑客们想要的可能不止是监测你的线上活动这么简单，他们还可以通过你的 IP 地址来获悉你的身份以及位置。这将使你容易遭受潜在恶意程序和其他类型的数字攻击的侵袭。

**安全使用公共 WiFi**：如今的公共免费 Wi-Fi 可谓是无处不在，对于旅途中的用户以及家中未安装 WiFi 网络的用户尤为吸引。尽管这听起来不错，但是在未隐藏你的 IP 地址的情况下使用免费 WiFi 会为你带来一系列安全隐患。免费且无保障的 Wi-Fi 网络无疑给了黑客们可乘之机，严重危及到了你的数据安全。

**规避组织性的线上限制**：你的工作场所或学校可能会在你使用他们的网络和 IP 地址的时候限制你访问某些网站。如果你可以隐藏 IP 地址，那么你就可以避开所有这些类型的限制。

### 2. Proxy 代理服务器

代理伺服器是 VPN 流行前，多数人会使用的“隐藏 IP”工具。匿名代理服务器是一种特殊类型的服务器，充当家庭网络和互联网其余部分之间的中介，所有流量都通过该中间服务器进行路由。所以你在 Internet 上访问的任何网站都只会看到代理服务器的 IP 地址，而不是你的实际 IP 地址。当信息从网站返回时，它首先通过代理服务器，然后再与你的设备进行通信。

尽管它可以像 VPN 一样掩盖你的 IP 地址，但与 VPN 相比，代理服务器的安全性较差。因为从你的设备发送到代理服务器上的数据未被加密。ISP，政府或第三方会有其他手段来获取你的数据。因此，使用代理服务器需要注意你的行为是否有违法风险。

互联网上有许多免费的匿名代理服务器。这些服务器大多具有带宽流量限制，或者可能随时跑路，如果你不是偶尔使用，建议只使用经过审查的代理服务器。

另外， 因为类似 tcp 这样的协议是有一个相互通信的握手过程 ，所以只使用一重代理是可以被网络运营商查出源 ip 的，因为你的信息要发出去必须有你在这个网络的 ip，一个网络数据中就包含你自己的 ip。那么可以使用多层的代理，这样对方查你时候要一个一个运营商去查，很麻烦。

### 3. 洋葱路由器（TOR）

Tor 是实现匿名通信的开源网络。其名源于「The Onion Router」的缩写“洋葱路由器”。使用者可通过 Tor 浏览器连接由全球志愿者自愿提供的 6000+个中间服务器，从而达到隐藏使用者的真实位置、避免网路监控及流量跟踪的目的。

Tor 使用者的网络活动较难追踪。Tor 还可以授予你访问正常浏览器无法访问的站点的权限。这些站点被称为“暗网”，它们的域名大多数都是随机生成的。此方法只适合用来免费隐藏个人 IP，因为 Tor 在经过多个服务器中转后速度会比较慢。

另外，由于 Tor 是一个可广泛访问，免费且匿名的网络，因此它已成为暗网犯罪活动的主要手段，包括非法交易，身份盗窃以及毒品和武器的交易。使用 Tor 很有可能被 ISP 发现并追踪，进而引来相关部门的注意。

### 4. 连接移动网路

由于你的手机数据是一个完全不同的系统，因此具有不同的 IP 地址范围，因此这也是一种快速简单更改 IP 的方法。

更改移动网络将同时改变你的 IP 地址，因为相应的 IP 地址是路由器在你访问网络时分配给你的。如果你的移动设备购买了数据套餐，那么你就可以创建自己的无线热点并使用此网络上网，而非你通常连接的网络。这同样能隐藏你的 IP 地址。

当然此方法对于笔记本电脑和台式机 IP 地址不太适用，只适合在极少数情况和紧急情况下使用，并且只能简单的改变 IP，不能达到隐藏身份的目的。如果你的网络活动比较敏感，或依赖桌面设备，尽量避免使用这种方法。

### 5. 使用公共 WiFi

由于 IP 地址是动态的，除非你注册了静态 IP 地址，否则随着位置的变化，IP 地址将随之改变。也就是说当你更换地区使用笔记本电脑，IP 都会根据你所在的位置而变化。因此，如果你需要另一种快速简便的方式来切换当前 IP 地址，那么使用公共 WiFi 就能帮你达成隐藏 IP 的目的。虽然此方法不能持续多次的使用，但的确是匿名和安全地上网的方法。

公共 WiFi 是开放式的网络，存在许多未经授权访问和破坏设备安全性的风险，因此在连接之前最好准备一定的保护措施，例如使用 VPN 等，不过 VPN 本身已经能够实现隐藏 IP 的目的。

### 6. 咨询你的互联网服务商

你可以要求你的互联网服务提供商（ISP）为你分配一个新的 IP 地址。 ISP 可以为你分配两种类型的 IP 地址。有一种静态 IP 地址，该地址是永久性的，除非你提出请求，否则不会更改。另一种是动态 IP 地址，该地址经常产生变化。

不过此方式只能更改你的 IP，却不能隐藏它，而且每次更换 IP 都需要与运营商客服取得联系，并没有太多的意义。

一般的黑客都是用类似肉鸡的方式，因为这样做水表是安全的，黑客只要藏好自己水表不被发现就是最好的了。把自己抓的肉鸡做成代理，比如 Windows 可以做成 VPN，Linux 可以做成 Socks5。一些比较敏捷的操作，比如 web 攻击，直接连接代理然后在本机操作就好，一些比较耗费时间，机器消耗比较大的可以直接丢到肉鸡上跑，比如暴力破解密码，批量扫描肉鸡等。

## 常见问题

### IP 地址是与你本人关联的吗？

不是。你的 IP 地址只会与每个位置相关联。你在家中连接付费互联网时使用的 IP 地址，与你在家查收电子邮件，在咖啡店等待咖啡，在办公室工作中的 IP 都是不同的。

### 你的 IP 地址会变化吗？

是的。即使你只在家中使用互联网，你家的 IP 地址也有可能发生变化。你可以联系你的互联网服务提供商（ISP）进行更改。此外，因互联网连接问题而重新启动调制解调器或路由器等常规操作也可能导致 IP 地址发生变更。

### 多台设备是否可以有相同的 IP 地址？

这是一个有点棘手的问题 — 答案是既是又不是。多台设备可以共享同一个外部（公共）IP 地址，但每个设备又都有自己的本地（私有）IP 地址。例如，ISP（互联网服务提供商）为你的家庭设置了一个外部 IP 地址。由于路由器是实际连接到互联网的，所以 IP 地址被分配给路由器。然后，路由器为每次连接到互联网的每台设备分配一个本地 IP 地址。外部 IP 地址是与外部世界共享的，而你的本地 IP 地址不会在你的私人家庭网络之外共享。

### IP 地址会被用完吗？

在互联网最初设计时，使用的是“第四版”地址。它使用 32 位（4 字节）地址，意味着我们最多可以有 42 亿个地址。这在当时似乎足够了，但在一个平均美国家庭拥有 11 台联网设备的世界里，显然是不够的。

我们现在已经有了第六版 IP 地址，每个地址有 128 位。遗憾的是，第四版和第六版不能直接相互通信，所以人们仍将需要第四版地址很长一段时间。

IPv4 地址不足的问题已经相当严重，特别是 5G 网络发展所产生万物互联的基础，任何智能设备终端均需要 IP 地址来实现网络的接入。

IPv6 便应运而生，IPv4 使用的是 32 位地址，最多能够使用的地址为 2^32；IPv6 使用的是 64 为地址，最多能够使用的地址为 2^128。大家不要小看了指数的叠加，IPv6 的地址数足够涵盖地球上的所有设备。
