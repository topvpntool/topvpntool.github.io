## Netflix 教程简介

该如何在中国完整解锁 Netflix ？我们都知道 Netflix 在中国大陆也叫网飞、奈飞，它是全世界最大的视频流媒体平台，在除中国大陆地区以外的所有国家和地区均提供视频点播服务。现主要业务是提供视频流播服务和影片发行制作，市值已超越 Disney，目前全球订阅用户已经达到 2.2 亿。

## Netflix 套餐与订阅

Netflix 付款是订阅制模式，绑定信用卡或者移动支付方式后，每月自动扣费，当然国内的奈飞用户主要都是去合租平台购买账号，相较而言更方便，也更便宜。

套餐等级一般有三种，分别支持最多一台设备 / 两台设备 / 四台设备同时在线，分别提供最高普通分辨率 / 高清 / 超高清分辨率。虽然各国家地区的套餐分区基本都是这三档，但各地的价格差距较大；比如这三档套餐在美区售价分别为 $8.99，$13.99， $17.99 美元，而在全球价格较便宜的阿根廷地区，售价约为 $4/月，$6/月，$9/月。

下面会详细教大家如何选择 Netflix 账户订阅的地区，以及看不同区的内容。

### 账号所在区 

**Netflix 账号所在区由第一次注册时的 IP 地址决定**。若第一次注册使用的是美国 IP 地址，这个 Netflix 账号的注册地就是美区的。账号所在区仅与每月需要支付的订阅费有关，与能看到的内容无关。如果要切换其他区域（比如切换到价格更便宜的区域），需要停止当前订阅，在本账单期终止后，清除浏览器 cookie，使用新地区的 IP 地址登陆并重新订阅，即可更换账号所在区。更改 IP 切换 Netflix 账号所在区可使用 VPN 来做到。

### Netflix 如何付费

Netflix 不同区的价格与支付方式不同，一般都支持 VISA / MasterCard 信用卡，中国发行的 VISA / MasterCard 多数都可以绑定成功，部分银行的信用卡可能绑定不成功，需要试一下才知道。

所以， 有一张非银联标志的全币信用卡即可订阅 Netflix 流媒体。Netflix 绑定支付方式时，会扣 1 美元或大约等值的当地货币用于验证此信用卡是有效的，之后会显示扣款失败或退款。如果不确定自己是否需要长期使用，可以先绑定支付方式开通尝试一下。如果不需要，请注意在第二个月扣款前取消订阅，否则会自动扣款。

## Netflix 内容分区与字幕问题

### IP 地址与所看内容 

Netflix 根据用户的 IP 地址来提供不同的内容，与账号所在区无关。如果你注册的是美区账号，走到世界各地，Netflix 提供的内容资源都会不同，但是你依旧需要使用美元支付你的订阅。如有多个区的代理，换一下 IP 地址，可看的内容就会发生变化。不建议在同一设备上频繁变动，会导致网址重定向过多， 观看记录出现异常 ，如出现这种情况，请清除浏览器 cookie。

### Netflix 中文字幕问题 

Netflix 各区对中文字幕的支持并不一致。**香港、台湾、新加坡、马来西亚有较多华人的地区几乎所有视频均支持中文字幕。**美区、日区等中文字幕较少，仅有新出的 Netflix 原创剧集会支持中文字幕；对于几年前的老剧，可能港区和美区都有视频资源，但是美区没有中文字幕。如果需要 Netflix 中文字幕，第一选择是将代理切换到上述支持中文字幕的地区，或者手动添加字幕 / 双字幕。外挂字幕的方式：

-   **方法一**：如需外挂字幕，可以直接按快捷键 CTRL + SHIFT + ALT + T 添加本地字幕文件。
-   **方法二**：安装 Super Netflix 插件 。两种方法一致，插件也只是免去记快捷键。使用外挂字幕时，字幕格式有限制。

## 为什么 Netflix 有区域限制

Netflix 目录因国家/地区而异的原因很简单：版权法。电影制片厂在不同的国家/地区签订了不同的合同，因此一个系列可能在一个国家/地区可用，但在另一个国家/地区不可用，甚至在 中国无法观看 Netflix。

另外还有一个原因就是本地广播权，某些国家/地区可能拥有某些节目的权利，因此不允许在 Netflix 上可以同时访问他们。香港的纸牌屋就是一个很好的例子——因为本地电视网络拥有 Netflix 节目的版权，所以它不能在线播放。

美国拥有最好的 Netflix 媒体库并不奇怪。由于大多数原创内容来自美国，因此在那里提供更多系列是有道理的。所以美国 Netflix 媒体库的节目数量比其英国多近 1,000 部。

使用付费 VPN 就在于稳定性高加上速度非常的顺畅，让你可以观看直播或 Netflix 影集不卡顿。相比免费 VPN，付费 VPN 的速度快出数倍，并且因为付费翻墙工具拥有非常多的主机、服务器以及定时维护，这会使得网络稳定性比免费的好很多。

### 2. Netflix 跨区成功率接近百分百

免费 VPN 无法完全隐蔽我们的 IP 地址，也无法把我们的 IP 准确的搬到其他国家，这就会导致无法正常跨区到美国、日本、加拿大、英国等等拥有与本地不同的 Netflix 媒体库的国家。而付费 VPN 如 PrivateVPN、ExpressVPN 等等，都可以让你一键实时联机到其他国家，而且地址也更准确和快速。

### 3. 确保隐私安全

VPN 可以确保我们使用网络的安全，透过 VPN 翻墙后浏览网站、观看影片或下载数据可以把自己的 IP 隐藏。隐藏自己的 IP 地址可以避免有心人士记录，因为他人可以经由 IP 查到你用网络的地方，这些地方可能是住家、学校、工作的地方等等。所以付费 VPN 不仅可以解除 Netflix 限制，同时也确保我们的安全

## 解锁 Netflix 教程中的常见问题

### FAQ1：什么网络才可以看 Netflix， 为什么我购买的 / 搭建的 XX 提示有代理 

Netflix 封禁了绝大部分来自 IDC 的 IP 地址。若购买 奈飞翻墙 VPN 服务，需要联系客服确认有可看 Netflix 的 streaming IP 类似服务，如果不想这么麻烦，可以直接选择我们测试中表现最好的 PrivateVPN。

另外有部分小众机场的 IP 地址可能没被列入 Netflix 的黑名单，但是购买风险、维护成本都不低，还是更推荐购买可靠的 VPN。

### FAQ2：我的 Netflix VPN 可以观看其他流媒体服务吗？

一般是可以的！我们为 Netflix 推荐的大多数 VPN 还可以与其他流行的流媒体服务一起使用，例如 Hulu、BBC iPlayer、Amazon Prime Video、Crave TV、ITV、All 4、Vudu 以及许多直播电视和体育流媒体服务。因为相对来说，Netflix 屏蔽 VPN 的措施最严格，如果 VPN 能够适用于 Netflix，那么大部分情况下也能适用于其他流媒体服务。

### FAQ3：如何下载 Netflix 客户端，为什么我搜不到 

Windows 商店需要切换到美区才能下载 Netflix 客户端。

iOS 设备的 app store 需要切换到美国等区域。

Android 设备 Google play 账号是非 中国区 一般都可以下载，可能存在设备型号未识别，root 设备无法下载的情况，可以前往国际第三方应用市场下载最新安装包。

Android 盒子设备可以前往 apkmirror 下载安装包后导入安装。

### FAQ4：为什么我的盒子 / 手机使用 Netflix 时提示错误码 

国产手机很多可能没有 DRM 认证。而未认证的设备，即使硬件上支持，也无法播放有版权的高清内容，防止高清资源被录制盗版出去。Netflix 在未进行 DRM 认证的设备上无法提供高清播放。

盒子必须经过 Netflix 认证才能正常安装和播放 Netflix。

### FAQ5：为什么我没法看高清的 Netflix

确定你的套餐是三款中最贵的，基础套餐可看 480P，标准版可看 1080P，高级版可看 4K。

确定你的网络速度够快。

Netflix 对能够播放高清 /4K 的设备 处理器有要求，具体请查看 官网帮助。

-   Windows 平台，请使用 edge 浏览器 或 Windows 10 APP；Chrome 最高仅支持 720P。
-   macOS 平台，请使用 Safari 浏览器；Chrome 最高仅支持 720P。
-   iOS 平台，较少出现此类问题。
-   Android 平台，可能没有 DRM 认证。
